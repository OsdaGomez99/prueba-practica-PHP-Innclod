# Gestión de Documentos
Para optar por la vacante de Desarrollador Web para Innclod

## Requisitos Previos
Asegúrate de tener instalados los siguientes componentes antes de comenzar:

- PHP >= versión 8.1
- Laravel >= versión 10.10
- Node >= 18.16.0
